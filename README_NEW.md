## рҹӣ пёҸ **Quick Start**

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)
- Git

### 1. Clone & Setup
```bash
git clone https://github.com/your-username/tellyousomeday.git
cd tellyousomeday

# Windows
setup.bat

# Mac/Linux  
chmod +x setup.sh
./setup.sh
```

### 2. Start Development
```bash
# Terminal 1: Start backend
cd backend
npm run dev

# Terminal 2: Start frontend
npm run dev
```

### 3. Open Your Browser
- Frontend: http://localhost:5173
- Backend API: http://localhost:5000
